# Fashion Trends MCP

This repo contains a Python FastMCP backend for fashion trend analysis and a Node.js bridge that renders ChatGPT UI widgets for those tools.

## Docs

- [Docs index](docs/README.md)
- [Backend server guide](docs/backend-server.md)
- [ChatGPT UI bridge guide](docs/chatgpt-ui-bridge.md)

## Main Paths

- `fashion_trends_mcp_server.py` - backend MCP server
- `chatgpt/server.js` - ChatGPT UI bridge
- `chatgpt/widget/app.html` - widget UI
- `scripts/` - optional setup, example, and connection helper scripts
